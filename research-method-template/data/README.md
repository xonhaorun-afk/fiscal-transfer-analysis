# 数据目录说明

将你的研究数据放在此目录下。

## 推荐的数据组织方式

### 方式一：按指标分文件（推荐）
```
data/
├── GDP.xlsx            # 每个Excel包含：区域 × 年份 的数据表
├── 人口.xlsx
├── 旅游收入.xlsx
└── 游客人次.xlsx
```

**Excel格式示例：**
| 省份 | 2024 | 2023 | 2022 | ... |
|------|------|------|------|-----|
| 北京市 | ... | ... | ... | ... |
| 天津市 | ... | ... | ... | ... |

### 方式二：按年份分文件
```
data/
├── 2018.xlsx           # 每个Excel包含：区域 × 指标 的数据表
├── 2019.xlsx
├── 2020.xlsx
└── ...
```

**Excel格式示例：**
| 区域 | GDP | 人口 | 旅游收入 | ... |
|------|-----|------|---------|-----|
| 区域A | ... | ... | ... | ... |
| 区域B | ... | ... | ... | ... |

### 方式三：单文件长表
```
data/
└── all_data.xlsx
```

**Excel格式示例：**
| 区域 | 年份 | GDP | 人口 | 旅游收入 | ... |
|------|------|-----|------|---------|-----|
| 区域A | 2018 | ... | ... | ... | ... |
| 区域A | 2019 | ... | ... | ... | ... |

## 注意事项

1. 区域名称必须与 `config.py` 中的 `REGIONS` 列表一致
2. 年份范围必须与 `config.py` 中的 `YEAR_START` / `YEAR_END` 匹配
3. 数据格式需与 `01_数据采集.py` 中的解析函数对应
4. 如有多种数据来源，可建立子目录分类存放
