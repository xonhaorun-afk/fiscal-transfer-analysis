# 研究方法复用指南

本模板项目提炼自「中央—地方财政转移支付效果分析」项目，将 **数据采集 → 指标计算 → 统计检验 → 可视化 → 仪表盘 → 报告生成** 的完整研究流程标准化，可快速复用到各类面板数据研究中。

---

## 🎯 适用场景

| 研究类型 | 示例 |
|---------|------|
| **区域经济分析** | 县域经济发展差异、乡村振兴成效评估 |
| **文旅产业研究** | 文旅资金绩效、游客人次时空分布、旅游收入影响因素 |
| **公共政策评估** | 专项资金分配效果、政策实施前后对比 |
| **社会事业分析** | 教育资源均衡、医疗资源配置、公共服务均等化 |
| **环境与资源** | 碳排放区域差异、生态补偿效果 |

## 🚀 快速开始（5步复用）

### 第1步：确定研究主题与核心问题
```
例：叙州区文旅产业发展分析
核心问题：
1. 各乡镇文旅发展水平差异如何？
2. 文旅投入与产出是否匹配？
3. 哪些因素影响文旅发展？
```

### 第2步：修改配置文件 [config.py](config.py)

需要修改的核心配置项：

| 配置区域 | 改什么 | 示例 |
|---------|-------|------|
| 项目基本信息 | 项目名称、副标题、研究问题 | 叙州区文旅发展分析 |
| 研究区域配置 | REGIONS列表、GeoJSON文件 | 叙州区各乡镇 |
| 时间范围 | YEAR_START / YEAR_END | 2018-2024 |
| 指标体系 | RAW_DATA_COLUMNS、INDICATOR_COLS | 旅游收入、游客人次等 |
| 区域分类 | CLASSIFICATION_COLUMN、阈值 | 按旅游收入分类 |
| 回归分析 | PANEL_REG_TARGET、解释变量 | 旅游收入影响因素 |
| 可视化配置 | 主指标、配色、标签 | 调整为文旅主题 |

### 第3步：准备数据

将Excel数据放入 `data/` 目录，支持两种组织方式：

**方式A：按指标分文件**
```
data/
├── GDP.xlsx
├── 人口.xlsx
├── 旅游收入.xlsx
└── 游客人次.xlsx
```

**方式B：按年份分文件**
```
data/
├── 2018.xlsx
├── 2019.xlsx
├── 2020.xlsx
└── ...
```

### 第4步：适配数据采集模块

修改 [01_数据采集.py](01_数据采集.py) 中的 `_parse_eps_file` 等解析函数，匹配你的Excel格式。

**三级回退策略**（可直接复用）：
- 模式A：读取本地真实Excel数据
- 模式B：在线API采集（如akshare）
- 模式C：内置估算数据（确保程序可运行）

### 第5步：运行全流程
```bash
pip install -r requirements.txt
python main.py
```

### 3. 数据处理 [02_数据处理.py](02_数据处理.py) ⭐ 核心方法

**核心方法论可直接复用，按需修改指标计算：**

| 模块 | 是否需要改 | 说明 |
|-----|-----------|------|
| 数据清洗 | 少量 | 列名适配 |
| 指标计算 | 较多 | 根据研究主题自定义 |
| 描述性统计 | 不用 | 通用 |
| 相关性分析 | 不用 | 通用 |
| ANOVA + 事后检验 | 不用 | 通用 |
| 面板固定效应回归 | 少量 | 变量名调整 |
| 分位数回归 | 少量 | 变量名调整 |
| 诊断检验（VIF/异方差/自相关） | 不用 | 通用 |
| 空间自相关（Moran's I） | 少量 | 区域名适配 |

**修改 `calculate_indicators` 函数即可替换指标体系。**

## 🔧 常见问题

**Q: Excel格式不一样怎么办？**
A: 修改 `01_数据采集.py` 中的解析函数，主要调整 `start_row`（数据起始行）和年份列的遍历逻辑。

**Q: 指标计算逻辑不同怎么办？**
A: 修改 `02_数据处理.py` 中的 `calculate_indicators` 函数，保留框架，替换计算公式。

**Q: 不需要空间分析怎么办？**
A: 在 `config.py` 中可以不配置GeoJSON，`02_数据处理.py` 会自动跳过空间自相关计算。

**Q: 不需要回归分析怎么办？**
A: 可以只运行 `--collect --visualize`，跳过处理步骤中的回归部分。

## 📚 核心方法论沉淀

本模板的核心价值在于以下**可复用的研究方法**：

1. **三级回退数据采集策略** — 真实数据 > 在线API > 估算数据，确保程序总能跑通
2. **面板数据标准化处理流程** — 清洗 → 插值 → 指标计算 → 分类
3. **完整的统计检验体系** — 描述性 + 相关性 + ANOVA + 面板回归 + 分位数回归
4. **规范的诊断检验** — VIF + 异方差 + 序列相关 + 空间自相关
5. **地图+统计的多维可视化** — 空间分布 + 时序趋势 + 相关性 + 回归结果
6. **交互式仪表盘** — ECharts 9图联动，年份滑块切换
7. **自动化文档生成** — 一键产出Word报告和PPT
8. **工程化保障** — 单元测试 + CI/CD + 日志记录
